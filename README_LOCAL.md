
# تعليمات تشغيل نظام PETROTEC محلياً (Local Setup)

## 1. العرض السريع للمدير (UI Demo - Mock Data)
لعرض الواجهة والشاشات بدون الحاجة لقاعدة بيانات، استخدم الكود الحالي كما هو:

1. تأكد من وجود `Node.js` مثبت على جهازك.
2. افتح الترمنال في مجلد المشروع.
3. نفذ الأمر: `npm install`
4. نفذ الأمر: `npm start`
5. سيفتح الموقع على `http://localhost:3000`.

*ملاحظة:* البيانات هنا وهمية (Mock Data) وتعود لأصلها عند تحديث الصفحة.

---

## 2. البدء في تطوير الباك إند (Real System)
لقد قمنا بإنشاء ملفات التأسيس للباك إند وقاعدة البيانات.

1. قم بتثبيت **Docker Desktop** على جهازك.
2. افتح الترمنال ونفذ الأمر التالي لتشغيل قاعدة البيانات:
   ```bash
   docker-compose up -d
   ```
3. يمكنك الدخول على لوحة تحكم قاعدة البيانات (pgAdmin) عبر الرابط: `http://localhost:5050`
   - البريد: `admin@petrotec.com`
   - كلمة المرور: `admin`

سيتم ربط كود NestJS بهذه القاعدة في الخطوات القادمة.
